1.1 PPG光学传感器的4+2阵列设计：心率监测的核心

什么是PPG（Photoplethysmography）光电容积脉搏波技术？

PPG是一种非侵入式的光学检测技术，通过向皮肤发射特定波长的光，检测血液容积变化引起的光吸收变化，从而获取心率、血氧、血压等生理参数。其工作原理基于Lambert-Beer定律：光在通过含有吸光物质的介质时，透射光强度会随吸光物质浓度呈指数衰减。

AZ Ring的创新：4+2传感器阵列

传统智能手环通常采用1-2个LED的单点检测设计，而AZ Ring突破性地采用了4个LED + 2个光电二极管的环形阵列设计：

  • 2个绿光LED（525nm）：用于心率检测。绿光对血红蛋白的吸收率最高，能有效检测到脉搏波动。功率：3mW，工作模式：脉冲式点亮（占空比5%）
  • 1个红光LED（660nm）：用于血氧饱和度(SpO₂)检测。红光穿透深度更深，能检测动脉血氧含量。功率：2.5mW
  • 1个红外LED（940nm）：用于深层血流检测和血氧计算。红外光穿透深度达3-4mm，可检测指骨下的血流信号。功率:2mW
  • 2个光电二极管（Photodiode）：呈对角线分布，实现差分检测，消除环境光干扰。灵敏度：0.45 A/W @ 525nm，响应时间：<1μs

💡 环形90度分布的工程意义

4个LED呈90度均匀分布在戒指内环，这种设计的优势在于：

  • 全方位覆盖：无论戒指如何旋转，始终有2-3个LED处于最佳检测位置
  • 信号冗余：多个传感器同时工作，通过算法融合提升信号质量30%
  • 自适应选择：系统自动选择信号质量最好的LED进行主检测，其他LED提供辅助
  • 功耗优化：根据佩戴状态动态启用LED数量，待机时仅启用1个LED，功耗降低60%

💡 为什么不用不锈钢或金？

  • 不锈钢：虽然便宜，但重量是钛的1.77倍，长时间佩戴会有压迫感。且含镍，10-15%人群会镍过敏

  • 铝合金：虽然更轻，但强度不足，且不耐腐蚀，汗液会导致表面发黑

2.2 PVD涂层工艺：耐磨、抗划、持久如新

什么是PVD（Physical Vapor Deposition）物理气相沉积？

PVD是一种在真空环境下，通过物理方法（蒸发、溅射）将金属材料蒸发成原子或分子，沉积在基材表面形成薄膜的技术。与传统电镀不同，PVD涂层与基材形成分子级结合，附着力极强，不会脱落。

AZ Ring的PVD涂层规格

  • 涂层材料：氮化钛（TiN）、类金刚石碳（DLC）、氮化铬（CrN）
  • 涂层厚度：2-5 μm
  • 硬度：莫氏硬度8-9级（仅次于蓝宝石和钻石）
  • 颜色选择：
    • 金色：TiN涂层（24K金视觉效果）
    • 黑色：DLC涂层（高科技感）
    • 玫瑰金：TiN + CrN复合涂层
    • 银色：原色钛（抛光处理）
  • 耐磨性测试：通过5000次Taber磨耗测试，涂层无明显磨损
  • 耐腐蚀性：盐雾测试1000小时无腐蚀
  • 环保性：不含重金属，符合RoHS标准

PVD工艺流程

  1. 预处理：超声波清洗 → 等离子清洗（去除表面油污和氧化层）
  2. 预热：真空腔加热至200-300°C
  3. 离子轰击：Ar⁺离子轰击表面，增加粗糙度，提升附着力
  4. 沉积：Ti靶材溅射 + N₂反应气体，形成TiN涂层
  5. 冷却：缓慢降温至室温
  6. 质检：膜厚测试、附着力测试、硬度测试

2.3 精密加工技术：微米级的工艺控制

CNC五轴加工：±0.01mm的精度

AZ Ring的戒身采用五轴联动CNC加工中心进行精密加工：

  • 设备：德国DMG MORI五轴加工中心
  • 精度：定位精度±0.005mm，重复定位精度±0.003mm
  • 刀具：硬质合金铣刀（直径0.5-3mm）
  • 转速：15000-25000 RPM（钛合金加工）
  • 进给速度：50-150 mm/min
  • 冷却：高压冷却液（100 bar），防止加工变形

加工难点与解决方案

  • 钛合金加工难度大：钛的导热性差，切削温度高达800°C，刀具易磨损
    → 解决：采用PCD（聚晶金刚石）涂层刀具，高压冷却
  • 薄壁零件易变形：戒身厚度仅0.26mm，加工时易变形
    → 解决：分层加工，多次轻切削，专用夹具固定
  • 内环曲面加工：内环需要光滑的弧形面，避免硌手
    → 解决：球头铣刀 + 多轴联动，实现R角过渡

内环抛光：Ra≤0.2μm的镜面效果

直接接触皮肤的内环需要达到镜面抛光效果（表面粗糙度Ra≤0.2μm）：

  • 机械抛光：使用羊毛抛光轮 + 氧化铝抛光膏（粒度5000#-10000#）
  • 电解抛光：在硫酸-甲醇电解液中进行电化学抛光，去除微观凸起
  • 超声波辅助：抛光过程中施加超声波振动，提升效率30%
  • 质检：使用表面粗糙度仪检测，确保Ra≤0.2μm

蓝宝石镜面：硬度9级的保护玻璃

AZ Ring采用合成蓝宝石作为传感器窗口：

  • 材料：单晶蓝宝石（Al₂O₃）
  • 硬度：莫氏硬度9级（仅次于钻石）
  • 厚度：0.4mm
  • 透光率：>85%（525nm绿光）
  • 切割工艺：激光切割 + 金刚石研磨 + 超声波清洗
  • 固定方式：医用UV胶粘接（固化后硬度HV 85，抗冲击）
  • 优势：高硬度（不易划伤）、高透光率（不影响PPG信号）、化学稳定性（不与汗液反应）

三、传感器嵌入式集成技术：从分立到一体

将传感器"塞进"戒指只是第一步，更大的挑战在于如何让这些传感器稳定工作、长期可靠、防水防尘。这涉及柔性电路板设计、芯片级封装、热管理、EMI/EMC设计、密封工艺等一系列系统工程问题。

3.1 柔性电路板（FPC）设计：在曲面上走线

为什么要用柔性电路板？

传统刚性PCB无法弯曲，无法贴合戒指的环形结构。柔性电路板（FPC）采用柔性基材（聚酰亚胺PI），可以弯曲、折叠，完美适配戒指曲面。

FPC的4层叠层结构详解

  • 基材：聚酰亚胺（PI）薄膜，厚度25μm，耐温-269°C至+400°C
  • 铜箔：电解铜箔，厚度18μm（1/2 oz），最小线宽/线距0.08mm/0.08mm
  • 覆盖层：PI保护膜，厚度25μm，起到绝缘和保护作用
  • 补强板：在贴装区域加FR4补强，提供刚性支撑，方便焊接
  • 总厚度：4层FPC总厚度≤0.18mm

弯曲性能测试

  • 弯曲半径：≤3mm（动态弯曲），≤1mm（静态弯曲）
  • 弯曲寿命：10万次弯曲测试（MIT弯曲试验），0断线
  • 抗拉强度：>150 N/mm（纵向），>120 N/mm（横向）

走线设计的关键技术

  • 差分对走线：I²C总线采用差分对走线，阻抗控制90Ω±10%，减少EMI干扰
  • 地线包地：信号线两侧走地线，形成"GND-Signal-GND"结构，屏蔽干扰
  • 过孔优化：采用盲孔、埋孔技术，减少过孔数量，提升信号完整性
  • 阻抗匹配：高速信号线（SPI, 8MHz）进行阻抗控制，避免信号反射

3.2 传感器封装与固定：芯片级封装的精密工艺

芯片级封装（CSP）技术

为了节省空间，AZ Ring采用晶圆级芯片尺寸封装（WLCSP）：

  • 封装尺寸：与芯片裸片尺寸相同，无额外封装体积
  • 球栅阵列（BGA）：底部焊球直径0.25mm，间距0.4mm
  • 焊接工艺：回流焊接，峰值温度260°C，时间<10秒
  • 优势：体积小（节省50%空间）、散热好、寄生参数小

医用环氧树脂封装

LED、光电二极管等光学器件采用透明环氧树脂封装：

  • 材料：医用级环氧树脂（USP Class VI认证）
  • 透光率：>95%（400-1000nm可见光/近红外）
  • 折射率：n≈1.5，与蓝宝石镜面匹配，减少反射损失
  • 固化方式：UV固化（365nm紫外光，功率5000 mW/cm²，时间30秒）
  • 硬度：固化后邵氏硬度D85，抗冲击
  • 耐候性：耐温-40°C至+85°C，耐湿85%RH不变黄

热膨胀系数匹配设计

不同材料的热膨胀系数（CTE）不匹配会导致温度变化时产生应力，造成脱焊、开裂：

  • 钛合金：CTE = 8.6 ppm/°C
  • 聚酰亚胺（PI）：CTE = 12-16 ppm/°C
  • 芯片（Si）：CTE = 2.6 ppm/°C
  • 解决方案：在芯片与FPC之间添加底部填充胶（Underfill），CTE = 30-50 ppm/°C，起到缓冲作用

振动与冲击测试

  • 振动测试：10-2000 Hz扫频，加速度10G，时间2小时，0失效
  • 冲击测试：半正弦波冲击，峰值加速度50G，脉宽11ms，3轴各3次，0失效

3.3 防水防尘结构设计：5ATM的密封工艺

5ATM防水等级详解

AZ Ring达到5ATM（50米防水）等级，符合ISO 22810:2010标准：

  • 5ATM定义：能承受5个大气压（约50米水深）的压力
  • 适用场景：日常洗手、淋浴、游泳、浅水浮潜
  • 不适用：深潜、热水澡（>45°C）、桑拿房（水蒸气压力大）

  • 整体密封：戒身分为上下两部分，通过O型密封圈 + 超声波焊接形成整体密封

  • 超声波焊接：焊接功率1500W，频率20kHz，焊接时间0.8秒，形成分子级结合
  • 充电接触点：采用弹簧顶针结构，接触点镀金（厚度3μm），外围有密封圈保护
  • 传感器窗口：蓝宝石镜面与戒身之间用医用硅胶密封（Shore A50，宽度0.3mm）

  • 静态压力测试：水深1.5米，持续30分钟，检查无渗水
  • 动态压力测试：模拟游泳，水深5米，手臂摆动频率120次/分钟，持续10分钟
  • 温度冲击测试：5°C冷水 ↔ 40°C热水，各浸泡5分钟，循环5次，检查密封性
  • 老化测试：在40°C/95%RH环境下存放1000小时后，重新进行防水测试

  • IP68定义：完全防止灰尘进入（IP6X）+ 长期浸水防护（IPX8）
  • 防尘测试：在滑石粉尘环境中放置8小时，拆解检查内部无粉尘

四、PPG传感器工作原理深度解析：从光到数据

PPG（Photoplethysmography）光电容积脉搏波技术是AZ Ring心率、血氧监测的核心。理解PPG的工作原理、信号处理算法、降噪技术，是理解智能戒指如何实现医疗级精度的关键。

4.1 光学检测原理：Lambert-Beer定律的应用

血液容积变化的光学检测

人体组织可以分为几层：皮肤 → 脂肪 → 肌肉 → 血管 → 骨骼。当LED发出的光照射到皮肤时，一部分光被吸收，一部分光被散射，一部分光穿透组织后被反射回来，被光电二极管接收。

心脏每次搏动，动脉血管的血液容积会发生周期性变化。血液多时，光吸收多，反射光少；血液少时，光吸收少，反射光多。通过检测反射光强度的周期性变化，就能获取脉搏波信号。

Lambert-Beer定律

光通过吸光介质时，透射光强度与吸光物质浓度、光程成指数关系：

I = I₀ · e-α·c·d

  • I：透射光强度
  • I₀：入射光强度
  • α：吸收系数（与波长相关）
  • c：吸光物质浓度（血红蛋白浓度）
  • d：光程（组织厚度）

由于心跳时血液容积变化 → 血红蛋白浓度变化 → 光吸收变化，因此接收到的光强度也随之周期性变化，形成PPG波形。

不同波长的穿透深度与应用

波长 颜色 穿透深度 主要检测参数 优势
525nm 绿光 1-2mm 心率 血红蛋白吸收率最高，信号强
660nm 红光 2-3mm 血氧（氧合血红蛋白） 氧合血红蛋白吸收率高
940nm 红外光 3-4mm 血氧（去氧血红蛋白） 去氧血红蛋白吸收率高

血氧饱和度（SpO₂）的计算原理

血氧饱和度 = 氧合血红蛋白 / （氧合血红蛋白 + 去氧血红蛋白） × 100%

通过测量红光（660nm）和红外光（940nm）的吸收率比值，可以计算出血氧饱和度：

SpO₂ = 110 - 25 × (ACred/DCred) / (ACir/DCir)

  • AC：PPG信号的交流分量（脉动部分，心跳引起）
  • DC：PPG信号的直流分量（基线部分，组织固有吸收）

4.2 信号采集与处理：从模拟到数字

信号链路

PPG信号从光电二极管到最终的心率数值，经过以下处理流程：

  1. 光电转换：光电二极管将光信号转换为微弱电流（nA级别）
  2. 跨阻放大：将电流信号转换为电压信号，放大10^6倍
  3. 滤波：去除高频噪声（>20Hz）和低频漂移（<0.5Hz）
  4. ADC采样：24位高精度ADC，采样率512Hz
  5. 数字信号处理：FFT频谱分析、峰值检测、心率计算

24位高精度ADC的意义

  • 动态范围：24位ADC的动态范围达120dB，可以同时处理强信号和弱信号
  • 分辨率：2^24 = 16,777,216个量化级别，最小可分辨信号为满量程的0.00000006倍
  • 低噪声：有效位数（ENOB）达20位，噪声仅-120dBFS
  • 优势：可以准确捕捉到心跳引起的微弱光强度变化（通常仅1-2%的幅度变化）

采样率的选择：512Hz

为什么选择512Hz采样率？

  • 奈奎斯特定理：采样率至少是信号最高频率的2倍。人体心率范围40-200 bpm（0.67-3.33 Hz），最高频率约3.33Hz，理论上7Hz采样即可
  • 实际考虑：PPG信号包含多次谐波（2倍频、3倍频），需要更高采样率才能还原波形细节
  • 512Hz的优势：
    • 可以准确捕捉PPG波形的上升沿、下降沿、重搏波特征
    • 便于FFT运算（512 = 2^9，FFT效率高）
    • 为后续的血压估算、动脉僵硬度分析提供数据基础

4.3 降噪与滤波算法：消除干扰的艺术

PPG信号的主要干扰源

  • 运动伪影（Motion Artifact）：手指运动导致传感器与皮肤接触不稳定，产生大幅度干扰，频率0.5-20Hz，与心率信号重叠
  • 环境光干扰：阳光、室内灯光进入传感器，产生50/60Hz工频干扰
  • 基线漂移：呼吸、体位变化导致的低频漂移（<0.5Hz）
  • 电磁干扰：手机、WiFi等射频信号的干扰

AZLG的多层降噪策略

1. 硬件层面

  • 差分检测：2个光电二极管分别检测信号，相减消除共模干扰（环境光）
  • 环境光检测：LED关闭时采样一次，测量环境光强度，后续减去此基线
  • 屏蔽设计：传感器周围用黑色不透光材料包裹，减少环境光进入
  • 接地设计：电路板地线包地，减少EMI干扰

2. 算法层面

  • 自适应滤波（Adaptive Filter）：
    • 原理：使用加速度计信号作为参考，估算运动伪影成分，从PPG信号中减去
    • 算法：最小均方（LMS）自适应滤波器
    • 效果：运动状态下心率准确率提升35%
  • 小波去噪（Wavelet Denoising）：
    • 原理：将信号分解为不同频率成分，去除高频噪声
    • 小波基：Daubechies 4（db4）
    • 分解层数：5层
    • 阈值策略：软阈值
  • 陷波滤波（Notch Filter）：
    • 去除50Hz/60Hz工频干扰
    • 带宽：±2Hz
    • 衰减：>40dB
  • 移动平均滤波：
    • 窗口长度：10个样本（约20ms）
    • 作用：平滑高频噪声

3. AI深度学习降噪

AZLG开发了基于卷积神经网络（CNN）的AI降噪算法：

  • 训练数据：10万+小时真实PPG信号（覆盖静息、运动、睡眠等场景）
  • 网络结构：1D卷积神经网络，深度16层
  • 输入：512个采样点的原始PPG信号 + 加速度信号
  • 输出：去噪后的干净PPG信号
  • 效果：运动状态下信噪比（SNR）提升25dB，心率准确率>95%
  • 部署：模型量化后部署在戒指的MCU中，实时推理

💡 为什么AI降噪比传统算法更好？

  • 自适应能力强：AI可以学习到不同人群、不同场景下的噪声特征，自动适应
  • 非线性处理：传统滤波器是线性的，而PPG噪声往往是非线性的，AI可以更好地分离
  • 端到端优化：从原始信号直接到心率结果，中间步骤都由AI优化，减少误差累积

    • 原理：葡萄糖分子在特定波长（1600nm, 2100nm）有吸收峰

    • 解决方案：增加红外LED（1600nm, 2100nm），提升ADC精度至32位，AI校准算法
    • 目标精度：±15 mg/dL（符合ISO 15197:2013）
  • 连续血压监测：
    • 技术路径：脉搏波传导时间（PTT）法
    • 原理：血压与动脉僵硬度相关，僵硬度与PTT相关
    • 需要数据：PPG波形 + ECG信号（需增加ECG电极）
    • 校准方式：定期用袖带血压计校准（每周1次）
    • 目标精度：±5 mmHg（符合AAMI/ESH/ISO标准）
  • ECG心电图功能：
    • 实现方式：戒指内环嵌入电极，双手接触形成I导联
    • 应用场景：房颤检测、心律不齐筛查、QT间期测量
    • 目标：获得FDA Class II医疗器械认证

中期规划（2027-2028）

  • 血液生化指标：
    • 血脂（总胆固醇、LDL、HDL）
    • 尿酸（痛风预警）

开启"预防医学"新时代
